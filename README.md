# 🎟️ Conference Ticket Generator

![Status](https://img.shields.io/badge/status-em%20desenvolvimento-yellow)

Uma aplicação web para gerar ingressos personalizados para um evento de tecnologia com uma experiência visual moderna e interativa.

> Transforme os dados do participante em um ticket estilizado em poucos passos, com upload de imagem, validação de formulário e persistência dos dados.

## ✨ O que o projeto faz

- Cria um ticket personalizado com nome, e-mail e usuário do GitHub
- Permite fazer upload de foto para compor o ingresso
- Exibe uma interface moderna inspirada em desafios de UI
- Armazena as informações em um backend PHP com MySQL
- Pode ser executado localmente com Docker

## 🛠️ Tecnologias utilizadas

- HTML5 e CSS3
- JavaScript para interação no frontend
- PHP com PDO
- MySQL
- Docker Compose

## 📁 Estrutura principal

- `index.html` — interface principal
- `assets/css/` — estilos do projeto
- `assets/js/` — interatividade do formulário e geração do ticket
- `assets/php/` — scripts de cadastro e conexão com o banco

## 🧩 Projeto

Este projeto foi desenvolvido como uma proposta prática de frontend e backend, unindo uma interface bonita com uma lógica simples de cadastro e geração de ticket.
